# Multi-Step Loan Application Form (React + TypeScript)

## 🚀 Технологический стек

* **React 19** (Vite) — для быстрой сборки и работы.
* **TypeScript** — для обеспечения надежности и типизации данных.
* **React Bootstrap** — для адаптивного и аккуратного интерфейса.
* **React Router Dom v6** — для навигации между шагами формы.
* **Context API** — для управления состоянием без лишней сложности (замена Redux).
* **Custom Hooks** — логика запросов вынесена в `useFetch`.

## 🧠 Логика проекта

Приложение разделено на 3 логических этапа:

1.  **Шаг 1 (Личные данные):** Сбор ФИО, телефона (с автоматической маской) и пола. Валидация обязательных полей.
2.  **Шаг 2 (Адрес и Работа):** Динамическая загрузка списка категорий мест работы с API и ввод адреса.
3.  **Шаг 3 (Параметры займа):** Интерактивный выбор суммы и срока займа с помощью ползунков (Range).
4.  **Финал:** Отправка всех собранных данных единым объектом на сервер и отображение результата в модальном окне.
